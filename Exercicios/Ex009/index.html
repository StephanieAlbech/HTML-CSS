Listas Simples ordenadas

Listas ordenadas são todas aquelas onde a ordem dos componentes é muito importante.

  1. Lista 1
  2. Lista 2
  3. Lista 3
  4. Lista 4
  5. Lista 5

A tag Ol possui um parâmetro type onde copnfighuramos o tipo de marcador da lista atual. As opçôes de valores para esse parâmentro são:

  1. 1 - Valor padrão. Cria listas numeradas. Ex: 1,2,4,5,...
  2. A - Cria listas alfabéticas em maiúesculas. Ex: A,B,C,D,....
  3. a - Cria listas alfabéticas em minúsculas. Ex: a,b,c,d,....
  4. I - Cria listasd com algarismos romanos em maiúsculas. Ex: I,II,III,IV,...
  5. i - Cria listas com algarismos romanosem minpuesculas. Ex: i,ii,iii,iv,...
